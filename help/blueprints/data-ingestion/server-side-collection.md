# 事件轉寄藍圖

事件轉送藍圖顯示如何從Experience Platform[！DNL]轉送使用Adobe Experience Platform Web和Mobile SDK收集的資料 [!DNL Edge Network]]到所要的目的地。 您可以根據在標籤屬性（舊稱 Launch）中設定的事件和規則轉寄從 SDK 彙集的所有原始資料或特定資料。

## 使用案例

* 使用單一集合標記從網路或行動裝置彙集資料，減輕用戶端瀏覽器及應用程式上的代碼權重。將彙集的資料傳播到各個端點以提供單一的資料彙集來源。
* 將彙集的資料轉寄到合作夥伴應用程式或資料儲存位置以根據彙集的資料建置見解與應用程式。

## 應用程式

* Adobe Experience Platform 資料彙集
