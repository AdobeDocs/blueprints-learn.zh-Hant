# 增加網站參與度

透過相關體驗，改善網站逗留時間、每個工作階段頁面數以及與網頁內容的互動。 此目標的中心在於提供情境式個人化Web體驗，讓訪客透過內容和轉換路徑持續參與和前進。

## 關鍵績效指標

下列KPI會衡量達成此目標的進度。

| KPI | 說明 |
| --- | --- |
| 頁面逗留時間 | 訪客在關鍵頁面上的平均逗留時間 |
| 參與度 | 包括點按數、捲動數和內容消耗數的互動複合分數 |
| 轉換率 | 完成所需動作的參與訪客百分比 |

| 圖樣 | 類別 | 如何支援此目標 |
| --- | --- | --- |
| [匿名訪客網頁Personalization](/help/blueprints/use-case-patterns/personalization/anonymous-visitor-web-personalization.md) | 個人化 | 根據工作階段行為將個人化網頁內容提供給未識別的訪客，以提高參與度 |
| [已知訪客網頁/應用程式Personalization](/help/blueprints/use-case-patterns/personalization/known-visitor-web-app-personalization.md) | 個人化 | 使用完整的設定檔資料為已驗證的訪客個人化網頁和應用程式體驗 |
