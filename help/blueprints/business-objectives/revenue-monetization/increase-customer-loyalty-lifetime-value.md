# 提高客戶忠誠度和期限值

透過忠誠計畫、獎勵和個人化參與，深化客戶關係並最大化長期價值。 此目標專注於建立持續的客戶關係，透過一致、價值導向的體驗，推動重複購買、更高的期限值及品牌宣傳。

## 關鍵績效指標

下列KPI會衡量達成此目標的進度。

| KPI | 說明 |
| --- | --- |
| 客戶期限值 | 客戶在整個關係中的總預測收入 |
| 保留 | 在定義期間內保持作用中之客戶的百分比 |
| 向上銷售/交叉銷售% | 購買其他或升級產品的現有客戶百分比 |
